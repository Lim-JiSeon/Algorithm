### 소스코드
+ 메모리 : 30864 KB
+ 시간 : 72 ms
```python
n = int(input())
A = list(map(int, input().split()))
B = list(map(int, input().split()))
answer = 0

for i in range(n):
    a = min(A)
    b = max(B)
    answer += a*b
    del A[A.index(a)]
    del B[B.index(b)]

print(answer)
```
