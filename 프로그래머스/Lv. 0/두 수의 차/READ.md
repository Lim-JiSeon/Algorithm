# 두 수의 차(사칙연산)

## 문제 설명
정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

## 문제 풀이
- minus 함수 만들기<br>
*minus(num1, num2) : num1과 num2을 빼는 기능 제공*
```python
def minus(num1, num2):
    return num1-num2

```
```javascript
function minus(num1, num2) {
    return num1-num2;
}
```

## 코드
### Python 코드
```python
def minus(num1, num2):
    return num1-num2

def solution(num1, num2):
    return minus(num1, num2)
```

### JavaScript 코드
```javascript
function minus(num1, num2) {
    return num1-num2;
}

function solution(num1, num2) {
    return minus(num1, num2);
}
```
